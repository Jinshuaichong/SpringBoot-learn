## 项目准备 
1. 启动虚拟机
2. 虚拟机终端运行systemctl run docker
3. 终端执行命令 sh /data/soft/restart.sh
4. vscode打开前端文件夹 并运行npm脚本
5. 启动nginx

## 微服务启动顺序推荐
1. GatewayApplication
2. SystemApplication
3. ContentApplication
4. MediaApplication
